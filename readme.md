## 1. Run Locally

Install dependencies

```bash
  mvn clean package
```

Start the server

```bash
  mvn spring-boot:run
```

## 2. Link Access swagger ui

```bash Project Service
http://localhost:8080/swagger-ui/index.html#/
```

```bash User Service
http://localhost:8090/swagger-ui/index.html#/
